CategoryList Response Elements
------------------------------

| Name           | Description                                               |
| :------------: | :-------------------------------------------------------- |
| *categoryList* | The container element for the list of category objects.   |
|                | Type: Container                                           |
|                | Ancestors: None                                           |
| *category*     | The name of the category.                                 |
|                | Type: String                                              |
|                | optional, may occur multiple times                        |
|                | Ancestors: categoryList                                   |
